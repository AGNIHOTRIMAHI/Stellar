# Stellar - AI-Enhanced Movie Ticket Booking System 🎬

Stellar is a premium, full-stack movie ticket booking application built using the MERN stack. It merges traditional booking functionality with modern AI capabilities and geolocation services to provide a high-end cinematic discovery experience.

---

> **Live Demo:** [stellar-frontend-mokx.onrender.com](https://stellar-frontend-mokx.onrender.com)
---

## ✨ Comprehensive Feature List

### 🧠 Intelligent Discovery & AI Integration
* **AI Trailer Insights:** Beyond just watching trailers, users can get AI-generated breakdowns and insights directly from the trailer content.
* **Hybrid Search:** Combines traditional metadata search with semantic context to help users find movies even with vague queries.
* **Smart Recommendations:** A recommendation engine that suggests movies based on genre preference and booking history.

### 📍 Seamless Booking Experience
* **Geolocation (Google Maps API):** Automatically detects the user's current city (e.g., Prayagraj) to show nearby theaters instantly.
* **Interactive Booking Calendar:** A custom-built, intuitive calendar interface allowing users to select specific dates for movie shows.
* **Real-time Theater Selection:** Dynamic dropdowns to select preferred theaters like PVR Vinayak based on the detected or selected city.

### 💳 Payments & Security
* **Razorpay Integration:** A fully functional payment gateway for secure, one-click ticket purchases.
* **Bloom Filter Authentication:** A high-performance security layer that uses Bloom Filters to optimize user authentication and reduce database overhead.
* **Secure User Profiles:** Managed user dashboards for viewing "My Bookings," account settings, and help center access.

### 🎬 Content Interaction
* **Integrated Trailer Player:** High-quality trailer streaming directly within the movie details page.
* **Comments & Reviews:** A social layer allowing users to read and leave feedback on the latest releases.

---

## 🛠️ Tech Stack

* **Frontend:** React.js, Tailwind CSS (Custom Dark Theme)
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **APIs:** Google Maps API (Geolocation), Razorpay (Payments)
* **AI Engine:** Gemini API 
* **Optimization:** Bloom Filters for Auth performance
