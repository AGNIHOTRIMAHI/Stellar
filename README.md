#🌌Stellar — AI-Powered OTT Streaming Platform ⭐⭐⭐⭐⭐

Stellar is a full-stack OTT streaming platform built to deliver a
modern, immersive movie and TV series experience. The platform
combines a sleek, responsive interface with intelligent, AI-driven
content recommendations powered by Gemini AI.

✅ This project is complete — having frontend development,
backend services, AI-based personalization, and cloud deployment.

---

## 🚀 Features

- 🎬 Modern OTT-style interface with responsive design
- 🎨 Real-time movie & TV metadata powered by TMDB API
- 🧠 AI-driven personalized recommendations using Gemini AI
- 🔐 Secure user authentication with JWT & password hashing
- ⚒️ RESTful backend API built with Express.js and MongoDB
- ⚛️ Modular and maintainable React component architecture
- ↻ Global state management using Zustand
- 🌐 Seamless frontend-backend communication via Axios
- 🚀 Fully deployed and production-ready

---

## 🧰 Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios
- Zustand
- Vite
- TMDB API

### Backend
- Node.js
- Express.js
- MongoDB & Mongoose
- JWT Authentication
- Bcrypt.js
- Postman (API testing)

### AI Integration
- Gemini AI (personalized, mood-aware content recommendations)

### Deployment
- Frontend & Backend deployed on Render
